## tomcat日志说明
* catalina.日期.log:   
  catalina引擎的日志信息，主要记录tomcat启动时信息，可看到启动时JVM参数信息等系统日志信息
* localhost.日期.log:   
  tomcat下的应用打印的日志信息(webapps下的应用)   
* localhost.access.日期.log:   
  访问日志，记录所有的web请求信息
* manager.日期.log:   
  存放webapps/manager项目的日志信息   
* host-manager.日期.log:   
  存放webapps/host-manager项目的日志信息
